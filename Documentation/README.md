# Quick Environment Tool

**One-Click World Generation for Unity**

Generate complete game environments instantly with different biome styles.

## Features

- **5 Environment Styles** - Forest, Desert, Snow, Tropical, Volcanic
- **4 Time Presets** - Dawn, Noon, Sunset, Night
- **4 World Sizes** - Small to Huge
- **One-Click Generation** - Complete worlds in seconds
- **Style-Appropriate** - Trees, water, and props match biome

## Installation

1. Import this package into your Unity project
2. Go to **Tools > Quick Environment**

## Usage

1. Select your environment style
2. Choose world size and time of day
3. Click "GENERATE WORLD"

## Requirements

- Unity 6 LTS or later
- Universal Render Pipeline (URP)
